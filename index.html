⚙️ Configuración de sincronización

Conecta con GitHub para que los datos actualizados sean visibles en todas las computadoras automáticamente.

Crea uno en github.com → Settings → Developer settings → Personal access tokens → Tokens (classic) con permiso repo
¿Cómo funciona?
Al subir un archivo, los datos se guardan en el repositorio de GitHub como data/ventas.json.gz. Cualquier computadora que abra el dashboard carga automáticamente los datos más recientes desde GitHub.
